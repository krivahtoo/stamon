## Features

- **Real-Time**: Keep track of your services and resources with up-to-the-second updates.
- **Lightweight**: Minimal resource usage, ideal for environments with limited resources.
- *More coming soon...*

# Build

## Frontend

```shell
cd ./frontend
pnpm build
```

## Back-end

```shell
cargo build
```
